简介

Markdown是一种轻量级标记语言,排版语法简洁,让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档,可与HTML混编,可导出 HTML、PDF 以及本身的 .md 格式的文件。因简洁、高效、易读、易写,Markdown被大量使用,如Github、Wikipedia等网站,如各大博客平台:WordPress、Drupal、简书等。

在一行中用三个及以上的星号(*)、减号(-)、等于号(=)或者下划线(_)来创建分隔线。

无序列表

无序列表可以在每行开头用星号、加号、减号来表示,也可以三者混合一起,推荐使用相同的字符,避免混乱。

  • 一朵百合花
  • 两朵百合花
  • 三朵百合花

有序列表

有序列表用数字接着一个英文句点来表示,数字可无序。

  1. 一朵百合花
  2. 两朵百合花
  3. 三朵百合花

插入代码

  • 插入单行代码,用一对反引号把代码引起来。
  • 插入多行代码,用一对 三个连续的反引号把多行代码引起来。

    网址链接

    网址链接有两种方式:行内式、参考式。
    对于行内式,链接文字是用 [方括号] 来标记,网址是用 (括号) 来标记,Title是用 “双引号” 来标记的。 这是一个行内式的网址链接:Google

对于参考式,一般写法是:(注意下面的代码中间需要空一行)

这是一个参考式的网址链接:Google

在参考式中,还有一种隐式写法是:
